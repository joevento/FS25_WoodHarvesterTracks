# Dirty Tire Tracks Mod
<i>Current version 1.0.1.1</i>

This mod adds extra tyre tracks to vehicles for more realistic behaviour in forested areas. It was originally created for heavy tracked machines, but the functionality has been extended to all wheeled and tracked vehicles.

The mod is enabled globally, but disabled per vehicle by default.

All credit goes to kenny456 for the original mod; I (DiscoFlower8890) only converted it to work with the latest version of the game.

Features:
- Enable/disable the mod for the current vehicle (the setting is saved to the config file and applies across all savegames)
- Enable/disable the mod globally for all vehicles
- Only administrators can change global settings in multiplayer
- All controls can be remapped in Input Bindings

Default controls:
- Left Alt + T — Toggle mod for current vehicle
- Left Ctrl + T — Toggle mod globally for all vehicles
